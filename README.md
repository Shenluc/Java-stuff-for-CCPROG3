Class Relationships
1. Association
a. Bidirectional - Object to Other Object
b. Directed - Object and other Object can't fight back
c. Reflexive - Object to itself or Object to same kind of object as itself.

2. Aggregation 
a. Object is independent from other Object's life cycle

3. Composition 
a. Object is dependent on other Object's life Cycle.

2&3 are Context sensitive so basically just check the specs on what you're supposed to have
